Приложение представляет из себя Ежедневник, который нацелен на ежедневное активное использование
## 781 строка кода
## 24 коммита


Принцип работы:
1. Пользователь открывает Приложение
2. Пользователь нажимает кнопку зарегестрироваться
3. Пользователь регистрируется и входит в аккаунт
4. Дальше выбор: перейти на окно календаря или погоды
    ----Погода----
    1. При входе пользователя в аккаунт сразу же делается запрос на сервер, поэтому данные о погоде будут уже выведены в виджетах

    ----Календарь----
    1.Пользователь может создать заметку:
        1. Выбрать дату на календаре
        2. Написать время в поле ниже
        3. Написать название события
        4. Создать
    2. При выборе события на поле событий, становятся доступны для нажатия кнопки редактирования и удаления записи
    3. Нажали кнопку редактирования:
        1. Открывается диалоговое окно с данными о событии, которые можно изменять
        2. При нажатии на кнопку Save/Cancel у пользователя спрашивается, сохранить ли изменения
        3. Окно закрывается, изменения принимаются
    4. Нажали кнопку редактирования:
        1. У пользователя просят подтвердить действие
        2. Если подтверждено, то запись удаляется
